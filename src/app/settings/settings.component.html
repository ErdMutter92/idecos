Settings

Debugger Mode

Boolean flag that denotes if the application is in debugging mode. Enables getting access to the chrome dev tools from the app, and console logs the action events to the console.

Auto Save Interval

Number that indicates the interval that the editor auto saves your unsaved actions. Number is in units of mins. Set to -1 to disable.

Scan Project At Start

Boolean flag that indicates that when the editor opens it should scan your entire project or just a directory at a time. Keeping this off will improve initial load of the application.
